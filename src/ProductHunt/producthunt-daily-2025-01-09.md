1. **创建 GitHub 存储库**：
   - 登录到你的 GitHub 帐户。

   - 将从 Webflow 导出的文件上传到新创建的 GitHub 存储库。确保文件结构符合 Shopify 主题的要求。

3. **添加 JSON 配置文件**：
   - 创建一个 `config/settings_schema.json` 文件，以定义主题设置和选项。

**步骤 4：将主题上传到 Shopify**

1. **登录 Shopify 后台**：
   - 进入你的 Shopify 商店后台。

2. **上传主题**：
   - 进入“在线商店” > “主题”。

3. **发布主题**：
   - 上传完成后，点击“发布”以使其成为商店的活跃主题。
